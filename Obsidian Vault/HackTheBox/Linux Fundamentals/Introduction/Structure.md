[[Linux Fundamentals]]
# Philosophy

1. Everything is a file
2. Small, single-purpose programs
3. Ability to chain programs together to perform complex tasks
4. Avoid captive use interfaces
5. Configuration data stored in a text file
# Components

- Bootloader
- OS Kernel
- Daemons
- OS Shell
- Graphic server
- Window Manager
- Utilities
